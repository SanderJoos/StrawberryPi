This page will teach you everything about the use of the API

please note that every API call except for getting all measurements will require basic authentication. You can use the username "Kemme" and the password "t" to authenticate.

Getting all measurements in the database

Getting all measurements currently in the database can be doing a get request through the following link: http://193.191.187.14:11378/StrawberryPiDataservice/API/measurement/. This will return all measurements in the database in JSON format

Adding a new measurement in the database

Adding a new entry to the database of measurements can be done by doing a put request trough the following link: http://193.191.187.14:11378/StrawberryPiDataservice/API/measurement/new. This will result in a call to the rest endpoints on the project that can be installed on the raspberry pi, that will give a new measurement which will be added to the database.

Deleting a measurement from the database

Param: long id

Deleting a measurement from the database can be done by doing a delete request through the following link: http://193.191.187.14:11378/StrawberryPiDataservice/API/measurement/delete-1. The 1 in the url is here an example for the parameter id, identifying which measurement has to be deleted. The full logical path without filled-in parameter is: http://193.191.187.14:11378/StrawberryPiDataservice/API/measurement/delete-{id}

Requesting all measurements before a given date

Param: date in stringformat. The format for the string is: yyyy-MM-dd

Getting all measurements before a given date can be done by the following URL: http://193.191.187.14:11378/StrawberryPiDataservice/API/measurement/before-{dateString}. {dateString} is the wanted date in the format: yyyy-MM-dd. This will initiate a get request that returns all the measurements before the given date in json format. An example: http://193.191.187.14:11378/StrawberryPiDataservice/API/measurement/before-2016-12-18
