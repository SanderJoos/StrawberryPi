Getting all temperaturemeasurements in the database

Getting all temperaturemeasurements currently in the database can be doing a get request through the following link: http://193.191.187.14:11378/StrawberryPiDataservice/API/temperature/. This will return all temperaturemeasurements in the database in JSON format

Requesting all temperaturemeasurements before a given date

Param: date in stringformat. The format for the string is: yyyy-MM-dd

Getting all temperaturemeasurements before a given date can be done by the following URL: http://193.191.187.14:11378/StrawberryPiDataservice/API/temperature/before-{dateString}. {dateString} is the wanted date in the format: yyyy-MM-dd. This will initiate a get request that returns all the temperaturemeasurements before the given date in json format. An example: http://193.191.187.14:11378/StrawberryPiDataservice/API/temperature/before-2016-12-18
