Getting all humiditymeasurements in the database

Getting all humiditymeasurements currently in the database can be doing a get request through the following link: http://193.191.187.14:11378/StrawberryPiDataservice/API/humidity/. This will return all humiditymeasurements in the database in JSON format

Requesting all humiditymeasurements before a given date

Param: date in stringformat. The format for the string is: yyyy-MM-dd

Getting all humiditymeasurements before a given date can be done by the following URL: http://193.191.187.14:11378/StrawberryPiDataservice/API/humidity/before-{dateString}. {dateString} is the wanted date in the format: yyyy-MM-dd. This will initiate a get request that returns all the humiditymeasurements before the given date in json format. An example: http://193.191.187.14:11378/StrawberryPiDataservice/API/humidity/before-2016-12-18
